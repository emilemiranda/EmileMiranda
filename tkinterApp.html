Wisconsin Cancer Analysis Desktop App

Created as a capstone project for UW-Madison's Cartography and GIS program, this desktop app was developed using Python. The tkinter and ArcPy libraries were leveraged to create a GUI where users can run IDW and OLS regression models to analyze the relationship between cancer rates and well nitrate levels in Wisconsin, producing both maps and regression reports with a simple point-and-click interface.
